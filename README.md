# Bank-Marketing-Data-Analytics
Data Analytics projects on Bank Marketing dataset - EDA and Logistic Regression


## About
Two data analytics projects analyzing Bank Marketing 
dataset to predict term deposit subscriptions.

## Projects

### Project 1 — Bank Marketing Inspection
- Loaded and explored 41,188 customer records
- Checked missing values and data types
- Performed summary statistics
- Visualized target variable distribution
- Created correlation heatmap

### Project 2 — Advanced Bank Term Deposit Analysis
- Age and job distribution analysis
- Call duration vs subscription analysis
- Contact method effectiveness
- Logistic Regression model — 91% accuracy
- Feature importance analysis

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Key Findings
- 88% customers did not subscribe (imbalanced data)
- Age group 30-40 are the most common customers
- Cellular contact gets more subscriptions than telephone
- Longer calls lead to more subscriptions
- Economy (cons.price.idx) is the strongest predictor
